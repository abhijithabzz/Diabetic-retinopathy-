This project focuses on the early detection of diabetic retinopathy using retinal fundus images and advanced machine learning techniques. It leverages deep learning models such as Convolutional Neural Networks (CNN) along with transfer learning approaches using pretrained models like DenseNet121, VGG16, and Xception to accurately classify the disease into five severity stages. The system incorporates image preprocessing techniques such as resizing and noise reduction, along with data augmentation methods to handle class imbalance and improve model generalization. Multiple models, including traditional machine learning algorithms like SVM and KNN, are also evaluated for comparison. Experimental results demonstrate that deep learning models, particularly Xception and DenseNet121, achieve high accuracy and better performance in detecting diabetic retinopathy. This automated system reduces manual effort, speeds up diagnosis, and can assist healthcare professionals in early detection and timely treatment, ultimately helping to prevent vision loss.
