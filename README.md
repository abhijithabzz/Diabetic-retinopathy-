This project focuses on the early detection of diabetic retinopathy using retinal images and machine learning techniques. It applies deep learning models such as Convolutional Neural Networks (CNN) along with transfer learning models like DenseNet121, VGG16, and Xception to classify the severity of the disease into different stages. The system uses preprocessing and data augmentation to improve model performance and accuracy. Experimental results show that deep learning models achieve high accuracy in detecting diabetic retinopathy, helping in faster diagnosis and supporting healthcare professionals in preventing vision loss.
